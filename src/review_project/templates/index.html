Приложение для проверки и улучшения кода (Тестовое задание для Skyeng)

Данное приложение предназначено для оптимизации процесса разработки, обеспечивая следующий функционал:

  • Автоматическая проверка кода на соответствие общепринятым правилам.
  • Отправка уведомлений пользователям о результатах проверки.
